## 📌 Project Overview

**Team Task Manager** is a collaborative full-stack web application designed to manage projects, teams, and workflows efficiently. It enables organizations and teams to streamline task assignments, track progress, and improve productivity through a centralized dashboard system.

The application supports **secure authentication, role-based access, task tracking, and project management features** for smooth team collaboration.

---

## 🚀 Features

### 🔐 Authentication & Security
- Secure login & signup system
- Role-based access control:
  - Project Manager
  - Team Member
- Session/JWT-based authentication (based on implementation)

---

### 📁 Project Management
- Create and manage multiple projects
- Assign team members to projects
- Track project progress in real-time

---

### 🗂️ Task Management
- Create tasks under projects
- Assign tasks to specific team members
- Update task status:
  - To Do
  - In Progress
  - Completed
- Delete and edit tasks

---

### 📊 Dashboard System
- Role-based dashboards
- Project overview and task distribution
- Progress tracking and visual summaries

---

### 👥 Team Management
- Add and manage team members
- Assign roles (Project Manager / Team Member)
- Monitor workload distribution

---

### 📈 Activity Tracking
- Track task updates and changes
- View recent activity logs
- Monitor team collaboration history

---

### 🎨 User Interface
- Clean and responsive UI
- Easy navigation dashboard
- Mobile-friendly design

---

## 🛠️ Tech Stack

- **Frontend:** React.js / Vite  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT / Session-based auth  
- **Styling:** CSS / Tailwind (based on implementation)

## ⚙️ Installation & Setup

```bash
# Clone repository
git clone https://github.com/PrathamPaul/Team-Task-Manager.git

# Move into project directory
cd Team-Task-Manager

# Install backend dependencies
cd Server
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Run backend server
npm start

# Run frontend app
npm run dev
